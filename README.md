# Leakage-Safe Cross-Cohort Survival Benchmarking Across TCGA BRCA, COAD, and LUAD

This repository contains the code and configuration for a leakage-safe cross-cohort survival benchmarking pipeline across TCGA BRCA, COAD, and LUAD.

Included contents:
- src/ survival benchmarking scripts
- configs/ configuration files
- requirements_lock.txt for environment reproduction

Note:
- Raw source data are not redistributed in this repository.
- Users should obtain TCGA/GDC source data from the official source and then run the pipeline locally.
